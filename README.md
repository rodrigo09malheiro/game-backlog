# 🎮 Game Backlog

App web para gerir a tua lista de jogos pessoal.

## 🔗 Links
- **App online:** https://game-backlog-eta.vercel.app
- **Repositório:** https://github.com/rodrigo09malheiro/game-backlog.git

## 🛠️ Tecnologias
- HTML, CSS, JavaScript
- RAWG API
- Vercel (CI/CD)

## ✨ Funcionalidades
- Pesquisar jogos via RAWG API
- Adicionar jogos ao backlog
- Gerir estados (A Jogar / Concluído / Abandonado)
- Filtrar por estado
- Dar nota pessoal (estrelas)
- Remover jogos
- Tema claro/escuro
